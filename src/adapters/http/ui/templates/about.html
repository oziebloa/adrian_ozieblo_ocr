About me and the project

My name is Adrian Oziębło and this project is a part of my master's
thesis and at the same time the first entry in my online portfolio.
My masters thesis is about comparing multiple machine learning methods
for text recognition in images. To do that, this OCR (optical character
recognition) application was created. Additionally in order to conduct
required experiment it's going to be deployed as a webservice in cloud
(kubernetes cluster) environment. The thesis itself is beeing written
in polish (since that is my native language) and it's available publicly
